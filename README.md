# UltraFast Downloader

A modern Windows GUI media downloader built with Python, CustomTkinter, and yt-dlp.

## Features
- Windows GUI
- MP3 download
- MP4 download
- Multiple URLs
- Progress bar
- Download queue

## Requirements
- Windows 10/11
- Python 3.10+
- FFmpeg (C:\ffmpeg\bin)

## Installation

```bash
pip install -r requirements.txt
python main.py
```
